# 2D-Graphics-Editor
A 2D graphics editor in C using character arrays to draw and manage shapes such as circles, rectangles, lines, and triangles.

Description

A simple 2D graphics editor developed in C using a 2D character array. The program allows users to draw and manage different shapes on a text-based canvas.

Features
Draw rectangle
Draw line
Draw triangle
Draw circle
Add objects
Delete objects
Modify objects
Display the picture
